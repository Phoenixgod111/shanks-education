# Структура контента предметов 📚

## Обзор

Каждый предмет имеет свою папку с подпапками для классов. Структура позволяет легко добавлять и редактировать учебный материал.

## Структура папок

```
web/subjects/
├── subjects-config.json          # Конфигурация всех предметов
├── russian/                      # Русский язык
│   ├── grade-5/
│   │   └── topics.json          # Темы для 5 класса
│   ├── grade-6/
│   │   └── topics.json          # Темы для 6 класса
│   └── ...
├── math/                         # Математика
│   ├── grade-5/
│   │   └── topics.json
│   └── ...
└── ...
```

## Формат файла topics.json

```json
{
  "grade": 5,
  "subject": "Русский язык",
  "topics": [
    {
      "id": "phonetics-5",
      "title": "Фонетика",
      "description": "Звуки и буквы русского языка",
      "difficulty": "easy",          // easy | medium | hard
      "estimatedTime": 30,           // минуты
      "lessons": [                   // уроки внутри темы
        {
          "id": "phonetics-intro",
          "title": "Введение в фонетику",
          "content": "Теоретический материал...",
          "type": "theory"           // theory | practice
        }
      ],
      "quiz": {                      // тест по теме
        "id": "phonetics-quiz-5",
        "questions": [
          {
            "id": "q1",
            "question": "Какие звуки называются гласными?",
            "options": ["а, о, у, и, ы, э", "б, в, г...", "..."],
            "correctAnswer": 0,
            "explanation": "Пояснение правильного ответа"
          }
        ]
      }
    }
  ]
}
```

## Добавление нового предмета

1. **Создать папку** для предмета: `web/subjects/new-subject/`
2. **Добавить в subjects-config.json**:
```json
{
  "id": "new-subject",
  "name": "Новый предмет",
  "icon": "school",
  "color": "#FF5722",
  "grades": [5, 6, 7, 8, 9],
  "description": "Описание предмета"
}
```
3. **Создать папки классов** и файлы `topics.json`

## Добавление новой темы

1. Открыть `topics.json` нужного класса
2. Добавить объект в массив `topics`:
```json
{
  "id": "new-topic-id",
  "title": "Название темы",
  "description": "Краткое описание",
  "difficulty": "medium",
  "estimatedTime": 45,
  "lessons": [...],
  "quiz": {...}
}
```

## Добавление уроков

В массиве `lessons` каждой темы:
```json
{
  "id": "lesson-unique-id",
  "title": "Название урока",
  "content": "Полный текст урока (HTML разрешен)",
  "type": "theory"
}
```

## Добавление тестов

В объекте `quiz` каждой темы:
```json
{
  "id": "quiz-unique-id",
  "questions": [
    {
      "id": "q1",
      "question": "Текст вопроса",
      "options": ["Вариант 1", "Вариант 2", "Вариант 3", "Вариант 4"],
      "correctAnswer": 1,           // индекс правильного ответа (0-3)
      "explanation": "Почему этот ответ правильный"
    }
  ]
}
```

## Система сложности

- **easy** - базовые понятия, повторение
- **medium** - углубленное изучение, применение
- **hard** - сложные задачи, анализ

## Инструменты разработки

- **ContentGenerator** - класс для генерации базовой структуры
- **SubjectManager** - управление загрузкой и отображением контента
